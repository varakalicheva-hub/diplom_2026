## О проекте

Работа посвящена автоматическому **аспектному анализу тональности** (aspect-based sentiment analysis, ABSA) русскоязычных отзывов об онлайн-курсах. Для каждого отзыва система извлекает квадруплеты вида **«аспектный термин — аспектная категория — мнение — тональность»** с помощью большой языковой модели (БЯМ) и промпт-инжиниринга, а результаты визуализируются в веб-приложении.

## Этапы, отражённые в репозитории

В репозитории представлена часть исследования — код и данные, относящиеся к сбору корпуса, автоматической разметке и оценке ее качества:

1. **Сбор корпуса** — парсинг русскоязычных отзывов об онлайн-курсах с сайтов-агрегаторов.
   Код: `notebooks/parsing_otzyvov.ipynb`

2. **Автоматическая разметка** — подключение к большой языковой модели через API и извлечение квадруплетов «аспектный термин — аспектная категория — мнение — тональность».
   Код: `notebooks/podkluchenie_k_byam_api.ipynb`
   Данные: `data/02_vyborka_dlya_razmetki_100.csv` → `data/04_avtomaticheskaya_razmetka_100.csv`

3. **Оценка качества разметки** — сравнение автоматической разметки с эталонной по двум подходам:
   - точное и частичное совпадение квадруплетов (precision, recall, F1) — `notebooks/ocenka_tochnoe_sovpadenie.ipynb`;
   - семантическая близость извлеченных фрагментов — `notebooks/ocenka_semanticheskaya_blizost.ipynb`.
   Данные: `data/03_etalonnaya_razmetka_100.csv` и `data/04_avtomaticheskaya_razmetka_100.csv`

Разработка промпта, сравнительный анализ нескольких БЯМ и веб-приложение Sentio в репозиторий не входят и описаны в тексте ВКР.


## Данные

Папка `data/` содержит корпус отзывов образовательной платформы «Нетология», использованный на этапе апробации. Файлы пронумерованы по порядку обработки.

| Файл | Описание | Кол-во отзывов |
|------|----------|:---:|
| `01_nachalnyy_dataset_300.csv` | Начальный датасет | 300 |
| `02_vyborka_dlya_razmetki_100.csv` | Выборка, отобранная для разметки | 100 |
| `03_etalonnaya_razmetka_100.csv` | Эталонная разметка выборки | 100 |
| `04_avtomaticheskaya_razmetka_100.csv` | Автоматическая разметка той же выборки моделью | 100 |

Эталонная и автоматическая разметки выполнены для одной и той же выборки из 100 отзывов, что позволяет сравнить качество автоматической разметки с ручной.

### Колонки

**Файлы без разметки (`01`, `02`):**

| Колонка | Описание |
|---------|----------|
| `review_id` | Уникальный идентификатор отзыва |
| `review_text` | Полный текст отзыва |
| `review_url` | Ссылка на источник отзыва |

**Файлы с разметкой (`03`, `04`):**

| Колонка | Описание |
|---------|----------|
| `review_id` | Уникальный идентификатор отзыва |
| `review_text` | Полный текст отзыва |
| `aspect_term` | Аспектный термин — упоминание аспекта в тексте |
| `opinion` | Оценочное выражение (мнение), относящееся к аспекту |
| `aspect` | Аспектная категория (например: «Материалы курса», «Команда сопровождения») |
| `sentiment` | Тональность оценки (положительная / отрицательная / нейтральная) |
| `review_url` | Ссылка на источник отзыва |

## Код

Папка `notebooks/` содержит Jupyter-ноутбуки (разработаны в Google Colab), реализующие основные этапы исследования.

| Ноутбук | Назначение |
|---------|------------|
| `parsing_otzyvov.ipynb` | Сбор (парсинг) отзывов об онлайн-курсах с сайтов-агрегаторов |
| `podkluchenie_k_byam_api.ipynb` | Подключение к большим языковым моделям через API на Python и автоматическая разметка отзывов |
| `ocenka_tochnoe_sovpadenie.ipynb` | Оценка качества разметки по строгому и частичному совпадению (precision, recall, F1) |
| `ocenka_semanticheskaya_blizost.ipynb` | Оценка качества разметки по семантической (косинусной) близости извлеченных фрагментов |

## Метод оценки качества

Качество автоматической разметки оценивается относительно эталонной по двум подходам:

- **Точное и частичное совпадение** — сопоставление предсказанных и эталонных квадруплетов с расчетом precision, recall и F1-меры в режимах строгого совпадения и частичного перекрытия.
- **Семантическая близость** — расчет косинусной близости векторных представлений автоматически извлеченных аспектных терминов и мнений по отношению к эталонным.
