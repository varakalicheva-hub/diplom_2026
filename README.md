## О проекте

Работа посвящена автоматическому **аспектному анализу тональности** (aspect-based sentiment analysis, ABSA) русскоязычных отзывов об онлайн-курсах. Для каждого отзыва извлекаются квадруплеты вида **«аспектный термин — аспектная категория — мнение — тональность»** с помощью большой языковой модели (БЯМ) и промпт-инжиниринга.

## Этапы, отраженные в репозитории

В репозитории представлена часть исследования — код и данные, относящиеся к сбору корпуса, автоматической разметке и оценке ее качества:

1. **Сбор корпуса** — парсинг русскоязычных отзывов об онлайн-курсах с сайтов-агрегаторов.
   Код: `notebooks/parsing_otzyvov.ipynb`

2. **Автоматическая разметка** — подключение к большим языковым моделям через API и извлечение квадруплетов «аспектный термин — аспектная категория — мнение — тональность».
   Код: `notebooks/podkluchenie_k_byam_api.ipynb`

3. **Оценка качества разметки** — сравнение автоматической разметки с эталонной по двум подходам:
   - точное и частичное совпадение квадруплетов (precision, recall, F1) — `notebooks/ocenka_tochnoe_sovpadenie.ipynb`;
   - семантическая близость извлеченных фрагментов — `notebooks/ocenka_semanticheskaya_blizost.ipynb`.

## Данные

### Корпус «Диплом 2026» — основной корпус исследования

Папка `data/korpus_diplom_2026/` содержит основной корпус русскоязычных отзывов об онлайн-курсах по нескольким тематическим направлениям, собранный с сайтов-агрегаторов «Сравни.ру» и «Отзовик». Файлы пронумерованы по порядку обработки.

| Файл | Описание | Кол-во отзывов |
|------|----------|:---:|
| `01_polnyy_korpus_2400.csv` | Полный корпус — все собранные отзывы | 2 400 |
| `02_vyborka_300.csv` | Выборка для автоматического анализа | 300 |
| `03_vyborka_dlya_ocenki_100.csv` | Выборка для оценки качества | 100 |
| `04_etalonnaya_razmetka_100.csv` | Эталонная (экспертная) разметка | 100 |
| `05_razmetka_gpt41mini_100.csv` | Автоматическая разметка моделью GPT-4.1 mini | 100 |
| `06_razmetka_deepseek_v32_100.csv` | Автоматическая разметка моделью DeepSeek V3.2 | 100 |
| `07_razmetka_yandexgpt_51pro_100.csv` | Автоматическая разметка моделью YandexGPT 5.1 Pro | 100 |

Файлы `04`–`07` содержат разметку одной и той же выборки из 100 отзывов: эталонную (ручную) и три автоматические, что позволяет сравнить качество работы трех больших языковых моделей между собой и с экспертной разметкой.

### Корпус «Нетология» — корпус апробации

Папка `data/netologiya/` содержит корпус отзывов образовательной платформы «Нетология», использованный на этапе апробации. Файлы пронумерованы по порядку обработки.

| Файл | Описание | Кол-во отзывов |
|------|----------|:---:|
| `01_nachalnyy_dataset_300.csv` | Начальный датасет — все собранные отзывы | 300 |

| `03_etalonnaya_razmetka_100.csv` | Эталонная (экспертная) разметка выборки | 100 |

Эталонная и автоматическая разметки выполнены для одной и той же выборки из 100 отзывов, что позволяет сравнить качество автоматической разметки с ручной.

## Код

Папка `notebooks/` содержит Jupyter-ноутбуки (разработаны в Google Colab), реализующие этапы исследования, представленные в репозитории.

| Ноутбук | Назначение |
|---------|------------|
| `parsing_otzyvov.ipynb` | Сбор (парсинг) отзывов об онлайн-курсах с сайтов-агрегаторов |
| `podkluchenie_k_byam_api.ipynb` | Подключение к большим языковым моделям через API на Python и автоматическая разметка отзывов |
| `ocenka_tochnoe_sovpadenie.ipynb` | Оценка качества разметки по строгому и частичному совпадению (precision, recall, F1) |
| `ocenka_semanticheskaya_blizost.ipynb` | Оценка качества разметки по семантической (косинусной) близости извлеченных фрагментов |

## Метод оценки качества

Качество автоматической разметки оценивается относительно эталонной по двум подходам:

- **Точное и частичное совпадение** — сопоставление предсказанных и эталонных квадруплетов с расчетом precision, recall и F1-меры в режимах строгого совпадения и частичного перекрытия.
- **Семантическая близость** — расчет косинусной близости векторных представлений автоматически извлеченных аспектных терминов и мнений по отношению к эталонным.
